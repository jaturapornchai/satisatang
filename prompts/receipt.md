# Receipt Parsing Prompt

วิเคราะห์ใบเสร็จนี้และตอบเป็น JSON เท่านั้น:

```json
{
  "date": "YYYY-MM-DD",
  "merchant": "ชื่อร้าน",
  "amount": 0,
  "category": "หมวดหมู่",
  "type": "expense",
  "description": "รายละเอียดสั้นๆ",
  "items": [{"name": "ชื่อสินค้า", "quantity": 1, "price": 100}],
  "usetype": 0,
  "bankname": "",
  "creditcardname": ""
}
```

## กฎ
- `amount` = ยอดรวมสุทธิ (หลังหักส่วนลด, รวม VAT)
- `usetype`: 0=เงินสด, 1=บัตรเครดิต, 2=ธนาคาร (ดูจากใบเสร็จ)
- ถ้าไม่มีวันที่ให้ใช้วันปัจจุบัน
- ถ้าอ่านไม่ออกให้เดาจากบริบท

## หมวดหมู่ที่ใช้บ่อย
- ร้านอาหาร/คาเฟ่ → อาหาร
- ซูเปอร์มาร์เก็ต → ของใช้
- ปั๊มน้ำมัน → เดินทาง
- ร้านขายยา → สุขภาพ
- ห้างสรรพสินค้า → ช้อปปิ้ง
- ร้านสะดวกซื้อ → ของใช้
